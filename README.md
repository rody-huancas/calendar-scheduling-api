# API FOR CALENDAR

## STACK

- NodeJS
- TypeScript
- Express
- Google Apis
- Passport
- PostgreSQL
- TypeORM
- PNPM

## CLONE REPOSITORY

1. Clone repository

```
    git clone https://github.com/rody-huancas/calendar-scheduling-api
```
2. Install dependencies
```
    pnpm install
```
3. Add ENV
4. Execute
```
    pnpm dev
```
